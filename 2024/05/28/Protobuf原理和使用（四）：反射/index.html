反射 API 允许程序在运行时检查和操作消息字段，而不需要了解字段的具体类型或名称

  1. **GetDescriptor()**：返回消息类型的描述符。描述符包含有关消息类型的所有信息，如字段名、字段类型等。返回google::protobuf::Descriptor类型的休想
  2. **GetReflection()**：返回消息的反射对象
