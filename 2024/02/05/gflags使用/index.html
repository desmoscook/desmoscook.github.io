gflags 是什么？

gflags 是一个由 Google 开发的命令行解析库。

gflags 主要用于解析命令行标志（flags），其极大的简化了从命令行读取参数并将其转换为变量的过程。

代码仓库：https://github.com/gflags/gflags

对于 C++ 项目，可以使用 cmake 或者从源码编译安装。

2.定义标志

标志 flags 是用于从命令行传递参数的变量，每个标志都有一个名称，类型，默认值和描述。

gflags 提供了诸多宏用于定义标志，如下表：

gflags 定义类型 描述
DEFINE_string string 类型
DEFINE_bool 布尔类型
DEFINE_int32 32 位整数
DEFINE_int64 64 位整数
DEFINE_uint64 无符号 64 位整数
EFINE_double 浮点类型 double
DEFINE_string C++ string 类型

gflags 还提供了 标志验证 的功能，用于在命令行参数解析后对标志的值进行验证，以确保其符合特定条件。

如果标志的值不符合要求，即标志验证函数返回 false，则会打印错误信息，并终止程序执行。
